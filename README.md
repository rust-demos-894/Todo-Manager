# Todo-Manager
玩具命令行Todo清单管理，本地存档
## Help：
> * TODO ===> 显示清单所有内容
> * ADD arg ===> 在清单末尾添加新task “arg”
> * UPDATE 3 ===> 更改序号为3的task，之后输入新内容
> * DEL 3 ===> 删除序号为3的内容，序号递补
> * CLEAR ===> 清空清单
> * EXIT ===> 退出程序，自动保存
> * 具有一定错误处理能力

## 之后可能会支持：
* 调整task序号
* 允许为task添加注释
* 支持更多种类的任务，不止task
* 为任务添加tag，允许分类显示
* 标记任务完成
* 为任务添加priority
